Tab5 BLE 設定

Web Bluetooth (Chrome / Edge 133+) から Tab5 の Wi-Fi / SSH / Tailscale を設定します。 通信は接続ごとに X25519 + AES-256-GCM で暗号化されます (OS ペアリング不要)。 プロトコルの詳細

設定すると、次のセッションから接続時に SHA-256(パスワード) を HKDF salt に折り込みます。 正しいパスワードを知らない端末は AES 鍵が一致せず、全ての暗号化された READ/WRITE が GCM tag mismatch で失敗します。

📟 tab5_claude_client Web Flasher

ブラウザから M5Stack Tab5 (ESP32-P4) にターミナルファームウェアを書き込みます。
Chrome / Edge など WebSerial 対応ブラウザ が必要です。

▶ GitHub リポジトリ ▶ リリース一覧 ▶ README (英) ▶ README (日)

注意: Tab5 出荷時の ESP32-C6 ファームウェアは esp_hosted 1.4.x で、本ファームウェア (2.x 系) と非互換です。 Wi-Fi を使う場合はリポジトリ README の「Wi-Fi C6 ファーム」セクションに従って先に C6 を更新してください。 Wi-Fi を使わずローカル UART 入出力のみで試す場合はそのまま書き込んで動作確認できます。
1. デバイス接続 / Connect
2. リリース選択 / Select release
3. 書き込み / Flash
4. シリアルモニタ / Monitor

フラッシュ後にデバイスのログを確認できます。Connect/Disconnect とは別系統で、 Flash で使った USB ポートを再利用します(一度認可済みなら再選択不要)。
